### ZeyFit Sunumu

ZeyFit kendimi geliştirmek için yaptığım bir projedir. Wissen Akademiden aldığım eğitimin sonucunda oluşturduğum bitirme projesidir.

##### Proje Hakkında Teknik Bilgiler

- Proje .NET C# Aspnet MVC CORE alt yapısıyla yazılmıştır.
- Entity Framework CORE - Code-First yaklaşımıyla yazılmıştır.
- Rol ve kullanıcı yönetimi Aspnet.Identity CORE ile yazılmıştır.
- FrontEnd işlemlerinde hazır temalar düzenlenerek kullanıldı.
- 5 katman (Entities,DAL,BL,PL) olarak yazıldı.
- Repository Design Pattern kullanıldı.
- Admin paneli bulunmaktadır.Yeni kullanıcı kaydı oluşturabilir, yeni bir aktivite ekleyebilir, yeni bir trainer girişi yapabılabilir.
- Ekran resimlerine ulaşabilirsiniz.
